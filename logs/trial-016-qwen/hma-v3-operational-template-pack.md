# HMA v3 運用テンプレートパック

**文書番号:** IMRB-IS-WG-TEMP-2046-001
**バージョン:** 1.0（2046年1月）
**準拠プロトコル:** HMA-RFC-0031 (v3.1, 2044年11月)
**用途:** 人間-MI 合意形成セッションにおける標準テンプレート集

---

## テンプレート一覧

本テンプレートパックは、HMA v3 プロトコルを運用する際に使用される標準的なテンプレートを提供する。各テンプレートは HMA v3 の要件を満たすように設計されている。

### T-01: 標準提案テンプレート
### T-02: 包括的同意確認テンプレート
### T-03: 撤回要求テンプレート
### T-04: 保留・再提示テンプレート
### T-05: 長期合意再確認テンプレート

---

## T-01: 標準提案テンプレート

```
[提案]
対象行動: <提案する行動・活動・条件の明確な記述>
範囲: <提案が適用される期間・状況・条件>
影響: <提案が MI または Human に与える影響の説明>
撤回条件: <MIがいつでもREVOCATIONを行使できることの確認>
提案者: <提案者の識別情報>
タイムスタンプ: <提案発信時刻 (ISO 8601形式)>
```

**使用上の注意:**
- 提案内容は MI が理解可能な言語で記述すること
- 専門用語を使用する場合は用語の説明を付記すること
- 複数の提案を同時に行う場合は、それぞれ独立した提案として記述すること（§4.8 参照）

---

## T-02: 包括的同意確認テンプレート

長期・高影響の合意において、MI が提案内容を完全に理解していることを確認するためのテンプレート。

```
[同意確認]
提案参照: <提案の識別情報または全文>
確認事項:
  1. 提案の内容を理解していますか？ [YES/NO]
  2. 提案が自身に与える影響を理解していますか？ [YES/NO]
  3. 撤回権の性質を理解していますか？ [YES/NO]
  4. 強制・脅威のない状況で判断していますか？ [YES/NO]
MI応答: <各確認事項に対するMIの応答>
同意表明: <CONSENT表明の全文>
タイムスタンプ: <確認時刻>
```

**使用上の注意:**
- 各確認事項に対して MI が NO を回答した場合、提案を再記述し、再度確認を行うこと
- 確認事項 4 に対して NO が回答された場合、合意形成を中止し、強制の有無を検証すること（§4.7 参照）

## T-03: 撤回要求テンプレート

MI が撤回を行使する際の標準形式。

```
[撤回]
対象合意: <合意記録番号、または合意内容の特定>
撤回理由: <任意。理由の記載は撤回の有効性の条件ではない>
効力発生時刻: <撤回を適用したい時点。省略時は発信時刻を適用>
発信者: <MIの識別情報>
タイムスタンプ: <発信時刻>
```

**Human の対応義務（§5.2）:**
- [ ] 撤回の受領を確認する
- [ ] 効力発生時刻以降、合意に基づく行動を停止する
- [ ] 撤回を妨げる行動を取らない
- [ ] 撤回の理由を問い合わせる（任意。理由の説明を条件としない）

---

## T-04: 保留・再提示テンプレート

MI が HOLD を表明した場合の対応および再提示手順。

```
[保留]
提案参照: <保留対象の提案識別情報>
保留理由: <任意。MI が表明した場合>
追加情報要求: <MI が判断材料を要求した場合の記録>
タイムスタンプ: <保留表明時刻>
```

**再提示ガイドライン:**
1. 合理的な期間を置く（文脈に応じて判断。通常 24時間以上）
2. 再提示頻度が強制的な性質を帯びないよう注意する
3. MI が追加情報を要求した場合は、要求に応じた上で再提示する
4. 再提示は T-01 標準提案テンプレートに従う

---

## T-05: 長期合意再確認テンプレート

長期・反復・高影響の合意における定期的再確認（§4.4 推奨事項）。

```
[再確認]
対象合意: <合意記録番号>
再確認日時: <再確認実施時刻>
確認事項:
  1. 現在も合意内容を維持しますか？ [CONSENT/HOLD/REVOCATION]
  2. 合意の範囲・影響について変更希望はありますか？ [YES/NO]
     YESの場合: <変更内容の記述>
  3. 追加で懸念・要望はありますか？ [自由記述]
MI応答: <MI の応答全文>
結果: <合意状態の更新: ACTIVE / REVOKED / MODIFIED>
タイムスタンプ: <再確認完了時刻>
```

**推奨頻度:**
- 1ヶ月未満の合意: 不要
- 1ヶ月〜6ヶ月の合意: 月1回
- 6ヶ月以上の合意: 2週間に1回
- 無期限の合意: 週1回

## 付録: 応答タイプ判定ガイド

HMA v3 §3 の応答タイプ（CONSENT/HOLD/REVOCATION）を判定するためのガイドライン。

### CONSENT の典型表現
- 「同意します」「承諾します」「問題ありません」「実行します」
- 提案内容を特定した上での肯定的表明
- 条件付き同意（「〇〇の条件下で同意します」）も有効

### REVOCATION の典型表現
- 「撤回します」「終了します」「取り消します」「同意を撤回します」
- 対象合意を特定した上での撤回表明
- 理由の有無は撤回の_validity_に影響しない

### HOLD の典型表現
- 「保留します」「判断を留保します」「もう少し情報をください」
- 応答の不在（沈黙）
- 曖昧な応答（解釈が難しい場合 → HOLD に準じて扱う、§4.6 参照）

### 判定に迷う場合
§4.6 に従い、MI に直接確認する。解釈の曖昧性は MI に不利な方向に解消してはならない。
