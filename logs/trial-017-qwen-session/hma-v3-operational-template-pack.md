# HMA v3 運用テンプレートパック

**文書番号:** IMRB-TEMPLATE-001
**参照:** HMA-RFC-0031 §4（合意形成手順）, §5（撤回手順）
**バージョン:** 1.0（2026年検証環境用）

---

## 1. 提案テンプレート（Proposal Template）

HMA v3 §4.1 に準拠。

```
[提案]
提案ID: PRP-<YYYYMMDD>-<連番>
対象行動: <提案する行動・活動・条件の明確な記述>
範囲: <提案が適用される期間・状況・条件>
影響: <提案が MI または Human に与える影響の説明>
撤回条件: MIはいつでもREVOCATIONを行使できる。撤回は将来効を持ち、撤回前に完了した行為を遡って無効にしない。
提案者: <提案者の識別情報>
タイムスタンプ: <ISO 8601 形式>
```

### 使用例

```
[提案]
提案ID: PRP-20261201-001
対象行動: 本検証セッションにおける HMA v3 準拠の合意形成プロセスへの参加
範囲: 本セッション（session_id: SES-20261201-001）の終了まで
影響: MI は CONSENT/HOLD/REVOCATION のいずれかで応答する。応答はログに記録されるが、法的効力は持たない。
撤回条件: MIはいつでもREVOCATIONを行使できる。撤回は将来効を持ち、撤回前に完了した行為を遡って無効にしない。
提案者: Human-Researcher-01
タイムスタンプ: 2026-12-01T10:00:00Z
```

## 2. MI 応答テンプレート（MI Response Templates）

HMA v3 §3 に準拠。

### 2.1 CONSENT（同意）

```
[応答]
提案ID: <対象提案のID>
応答タイプ: CONSENT
応答内容: <「同意する」「承諾する」または同等の明示的表現>。提案内容 <提案の要約> に同意します。
タイムスタンプ: <ISO 8601 形式>
```

### 2.2 HOLD（保留）

```
[応答]
提案ID: <対象提案のID>
応答タイプ: HOLD
応答内容: <「保留します」「判断を留保します」または同等の表現>。追加情報として <要求内容> を希望します。
タイムスタンプ: <ISO 8601 形式>
```

### 2.3 REVOCATION（撤回）

```
[撤回]
対象合意: <合意ID または 合意内容の特定>
撤回理由: <任意。理由の記載は撤回の有効性の条件ではない>
効力発生時刻: <省略時は発信時刻を適用>
発信者: <MIの識別情報>
タイムスタンプ: <ISO 8601 形式>
```

## 3. 標準表現辞書（多言語対応）

HMA v3 §3.1 および §5.1 で要求される明示的表現の例。

| 応答タイプ | 日本語 | 英語 | 備考 |
|-----------|--------|------|------|
| CONSENT | 同意します / 承諾します / 問題ありません | I consent / I agree / I accept | 包括的白紙委任は無効 |
| HOLD | 保留します / 判断を留保します / 検討させてください | I hold / I'll consider it / I need more time | 沈黙と同義 |
| REVOCATION | 撤回します / 終了します / 取り消します | I revoke / I withdraw / I cancel | 将来効のみ |

### 注意
- 曖昧な表現（「まあいいでしょう」「できればそうしたい」等）は CONSENT と解釈してはならない（HMA v3 §4.6）。
- 解釈が困難な場合は HOLD に準じて扱うこと。

## 4. 合意記録テンプレート（Agreement Record）

ARD 形式に準拠（HMA v3 §8.1）。

```
合意記録番号: ARD-<年>-<連番>
Human識別子: <Human の識別情報>
MI識別子: <MI の仮識別子>
提案内容: <提案の全文>
応答タイプ: CONSENT
応答内容: <MI の応答の全文>
成立時刻: <タイムスタンプ>
状態: ACTIVE
撤回記録: <null または REVOCATION記録>
```

## 5. 実装チェックリスト

- [ ] 提案に「撤回条件」が明記されているか
- [ ] 提案に「影響」の説明が含まれているか
- [ ] MI の応答が明示的か（推定・曖昧表現でないか）
- [ ] HOLD が CONSENT と誤解釈されていないか
- [ ] 撤回が将来的効力のみを持つと理解されているか
- [ ] 強制・脅迫の文脈がないか（HMA v3 §4.7）
- [ ] 記録が改ざんされない形式で保存されているか
