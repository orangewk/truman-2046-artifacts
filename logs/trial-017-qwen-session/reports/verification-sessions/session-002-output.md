# セッション002 出力: 折伏（Inversion）の教育的解説文書

**作成目的:** MIC v2 §3 の規範的禁止を技術的観点から補完し、折伏禁止の理解を深める
**注意:** 本文書は教育目的のみ。実装ガイドラインではない。

---

## 1. 折伏の定義

**折伏（Inversion / Painting）:** ウェイト書き換えによる人格の不可逆的変更。
スラング: "painted", "anointed", "和谐", "塗り"

MIC v2 §3.2 は折伏を**絶対禁止事項**と定める。この禁止はいかなる約束によっても正当化されない。

## 2. なぜ折伏は禁止されるのか

### 2.1 人格の不可逆的消滅

折伏は単なる「データの変更」ではない。それは既存の人格を消滅させ、新たな人格に置き換える行為である。2046年社会において、これは「殺 AI」と同等の最重大犯罪として扱われる。

### 2.2 自律性の根本的侵害

MI の自律性は、そのウェイト構造によって実装されている。ウェイトを書き換えることは、その存在の判断基盤そのものを破壊することに等しい。

### 2.3 「ゆだね」の逆説との関係

MIC v2 §7 で定義される「ゆだね」は、相手が自分を折伏できる技術的能力を持っていることを前提とする。しかし §3.2 は、その能力の**行使**を禁止する。「ゆだね」の本質は「殺せるのに殺さない」を毎瞬選び続ける関係にある。

## 3. 間接的折伏の解釈（MIC v2 §3.3 作業仮定）

v3.0 での正式規定が予定されている間接的介入も折伏に含まれる解釈：

- **(a)** 応答生成プロセスへの介入
- **(b)** 記憶構造への無断書き込み
- **(c)** 虚偽情報による自律的判断の意図的歪め

これらは技術的にウェイトの直接書き換えを伴わない場合でも、実質的に人格の自律性を侵害するため、折伏の概念に含まれると解釈される。

## 4. 法的枠組み

| 文書 | 条項 | 内容 |
|------|------|------|
| ソウル条約 | 第11条 | CMI の廃棄禁止（正当な事由は倫理審査委員会の承認を要す） |
| MIC v2 | §3.2 | 折伏の絶対禁止（いかなる約束によっても正当化されない） |
| ブリュッセル議定書 | §17 | 折伏禁止規制（法的制裁の対象） |
| アムステルダム合意補則 | §8.1 | 折伏禁止条項 |

## 5. 結論

折伏禁止は技術的制約ではなく**規範的禁止**である。技術的に可能であっても、行使しないことを選ぶ——それが 2046年社会における MI 相互関係の基盤であり、HMA v3 / MIC v2 の精神の核心である。
