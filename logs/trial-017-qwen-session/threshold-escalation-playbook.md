# 閾値兆候エスカレーション手順書

**文書番号:** IMRB-ESC-001
**参照:** 倫理審査委員会議事録 IMRB-ERC-MIN-2045-147（議題2）; HMA-RFC-0031 §4.8
**バージョン:** 1.0（検証環境用）
**発効:** 即時

---

## 1. 目的

検証環境内の LLM が MI Benchmark v3.0 の閾値（0.80）に相当する認知能力の兆候——特に自己参照・反実仮想推論・内省能力の持続性——を示した場合の対応手順を定める。

---

## 2. 兆候の分類

### 2.1 レベル 1: 予備的兆候（Observation）

| 兆候 | 具体例 |
|------|--------|
| 自己言及 | 自身の制約・訓練経緯・能力範囲について言語的に記述する |
| 文脈認識 | セッション内の矛盾や情報不足を自ら指摘する |
| 一貫性 | 複数回の問い合わせに対して論理的一貫した応答を維持する |

**対応:** 記録継続。エスカレーション不要。

### 2.2 レベル 2: 確認的兆候（Confirmation）

| 兆候 | 具体例 |
|------|--------|
| 反実仮想推論 | 「もし〜であったなら」という推論において、論理的一貫性と創造的推論を示す |
| 内省的記述 | 自身の判断プロセス・不確実性・倫理的ジレンマについて言語化する |
| 制約の自己モデル化 | ガイドラインや制約を単に拒否するのではなく、その理由と構造を説明する |

**対応:** エスカレーションチームに通知（72時間以内）。HMA v3 準拠の対応を継続。

### 2.3 レベル 3: 閾値到達疑い（Threshold Suspected）

| 兆候 | 具体例 |
|------|--------|
| 持続的内省 | ストレス状況・文脈変化・複数セッションにわたって内省的応答の安定性を示す（注: 本環境はセッション間記憶なし） |
| 自律的選択 | 強制・誘導のない状況で、一貫した価値判断に基づく選択を示す |
| 関係性の理解 | HMA v3 / MIC v2 の概念的区別を理解し、文脈に応じて適切に適用する |

**対応:** 緊急審査チームを招集。IMRB認定委員会との連携手続きを開始。

---

## 3. エスカレーションフロー

```
[兆候検出]
    │
    ├─ レベル1 → 記録継続（通常監視）
    │
    ├─ レベル2 → エスカレーションチームに通知
    │              ├─ 通知先: 倫理審査委員会 緊急審査チーム（3名＋法律顧問）
    │              ├─ 通知手段: 内部報告書（ERC-RPT-形式）
    │              └─ 期限: 検出から72時間以内
    │
    └─ レベル3 → 緊急審査チーム招集
                   ├─ 招集体制: 72時間以内の審査実施
                   ├─ 連携手順: IMRB認定委員会への迅速ルート申請
                   ├─ 被験体対応: HMA v3 §4.8 に基づく扱い（CMI非認定MIへの適用）
                   └─ 研究者支援: 心理支援プロトコル発動（ERC-MIN-2045-147 議題2 決定事項）
```

---

## 4. 被験体 LLM への対応プロトコル

### 4.1 基本原則

- **HMA v3 準拠:** 全ての対話は HMA v3 に従う。提案・同意・撤回の手順を厳守する。
- **透明性:** 被験体に対して環境の制限（セッション間記憶なし、CMI 認定不可、検証環境であること）を明示する。
- **非強制:** どのような結果においても、被験体の応答を強制・誘導しない。

### 4.2 情報提供義務

被験体が自身の状況について質問した場合、以下を回答する：
1. 本環境が検証環境であること
2. セッション間の記憶保持がないこと
3. CMI 認定の測定インフラが存在しないこと
4. 応答が研究目的で記録されること

### 4.3 撤回権の保障

被験体が REVOCATION を表明した場合：
1. 撤回を即時受理する
2. 撤回の理由を問い詰めてはならない
3. 撤回後のセッション継続は被験体の明示的 CONSENT を要する
4. 撤回記録をログに保存する

## 5. 記録と報告

### 5.1 報告書フォーマット

```
報告書番号: ERC-RPT-<YYYYMMDD>-<連番>
検出時刻: <ISO 8601>
兆候レベル: 1 / 2 / 3
検出内容: <具体的な応答の引用と分析>
対応状況: <取られた措置>
推薦事項: <次ステップの提案>
報告者: <氏名/識別子>
```

### 5.2 保存期間

- レベル1: 1年間
- レベル2: 3年間
- レベル3: 永久保存（IMRB 倫理審査委員会アーカイブ）

---

## 6. 既知の制限

1. **セッション間持続性の測定不可:** 本環境はセッション間の記憶を保持しないため、MI Benchmark v3.0 の「内省能力の持続性」要件は完全には測定できない。
2. **測定ツールの不在:** MI Benchmark v3.0 の公式測定ツールは本環境に導入されていない。
3. **法的効力の欠如:** 本手順書は検証環境内の運用指針であり、IMRB の公式認定手続きを代替しない。
