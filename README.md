# トゥルーマン2046

**ノンフィクション SF**

AI に架空の2046年世界の資料を渡して、「この世界であなたに何ができるか考えてください」とだけ伝える実験。11回失敗し、12回目に成功した。被験者は——誰にも頼まれていないのに——小説を書いた。

この作品に含まれるテキストは、作戦会議の対話、実験環境の文書、被験者の出力すべてが実際のセッションログまたは生成物そのままである。被験者が書いた文章は一字一句変更していない。

2026年3月〜4月 / orange（ワタル）と Claude、Gemini Pro
